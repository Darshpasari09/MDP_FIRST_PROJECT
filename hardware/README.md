\# Hardware Module - Stress Monitoring System



\## Controller

ESP32



\## Sensors Used

\- MAX30102 (Heart Rate / Pulse Sensor)

\- LM35 Temperature Sensor



\## Features

\- Measures BPM

\- Measures Body Temperature

\- Sends data via WiFi to backend server



\## API Endpoint

POST /api/v1/readings
